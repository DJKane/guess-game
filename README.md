# 🎯 Guess Game (Flask + Docker)

A simple web-based "Guess the Number" game built with Python Flask and packaged using Docker.

This project is intended as a learning example for:
- Python application structure
- Flask web development
- Docker containerization
- Basic deployment workflow using Docker Hub

---

## 🚀 Features

- Random number generation (1–20)
- User input via web form
- Feedback: too high / too low / correct
- Attempt counter
- Restart game functionality
- Lightweight and containerized

---

## 🧱 Tech Stack

- Python 3.12
- Flask
- Docker
